# Number-Guessing-Game
Number Guessing Game (Python)

This is a simple and fun command-line number guessing game written in Python. The program randomly selects a number between 1 and 100, and the player must guess it with feedback provided after each attempt.

How It Works:

  The computer randomly picks a number.

  The player enters guesses until they find the correct number.

  After each guess, the program tells the player whether the guess is:

  Too high

  Too low

  Out of range

When the correct number is guessed, the program displays the total number of attempts.

Features:

  Random number generation

  Input validation to prevent invalid or non-numeric entries

  Unlimited attempts until the correct guess

  Clean and readable Python code

Usage:

Run the script with:

python number_guessing_game.py


Follow the on-screen prompts to play.

Perfect For:

  Python beginners

  Learning loops, conditionals, and input handling

  Simple game and logic practice
